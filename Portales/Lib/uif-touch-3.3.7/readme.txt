- Dentro del header de su layout incluir los estilos:
  ej.
  <link href="PATH-HACIA-EL-FRAMEWORK/vendors/css/vendors-3.3.6.css" rel="stylesheet" type="text/css" />

  Si usa el tema default incluir:
  <link href="PATH-HACIA-EL-FRAMEWORK/uif-touch-3.3.6-default.min.css"" rel="stylesheet" type="text/css" />

  O si usa el tema dark incluir:
  <link href="PATH-HACIA-EL-FRAMEWORK/uif-touch-3.3.6-dark.min.css"" rel="stylesheet" type="text/css" />



- En su html (antes de cerrar el tag body) incluir el script de vendors y el del tema utilizado en el archivo de layout
  ej. 
  <script src="PATH-HACIA-EL-FRAMEWORK/vendors/js/vendors-3.3.6.js"></script>
  
  Si usa el tema default incluir:
  <script src="PATH-HACIA-EL-FRAMEWORK/uif-touch-default-3.3.6.min.js"></script>
  
  O si usa el tema dark incluir:
  <script src="PATH-HACIA-EL-FRAMEWORK/uif-touch-dark-3.3.6.min.js"></script>
